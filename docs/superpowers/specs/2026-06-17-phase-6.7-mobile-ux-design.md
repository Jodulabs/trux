# Phase 6.7 ‚Äî Mobile UX: from toy to instrument

*Branch: `phase-6.7-mobile-ux` ¬∑ Companion: [roadmap](../../../docs/2026-06-16-trux-roadmap.md)*

---

## Goal

The pocket-dev loop works (Phase 6.6), but live use exposed the UI as childish and several
functional gaps. Redesign the mobile UI to feel like a developer's instrument, and fix the
ergonomics surfaced by actually driving Claude from the phone.

---

## Aesthetic direction

A **refined dark developer's field console** ‚Äî not a chat toy:

- **Palette:** deep ink surfaces, layered (`--ink`/`--surface-1..3`), a single **warm copper accent**
  (`#e8843d`) ‚Äî deliberately not the childish blues. Cool slate for user bubbles to contrast the warm.
- **Type:** **IBM Plex Sans** (prose) + **IBM Plex Mono** (commands/code/labels) ‚Äî engineered,
  characterful. **Bundled locally** via `@fontsource` (no Google CDN ‚Üí stays sovereign + works offline).
- Real **SVG icons** (`Icon.tsx`), not emoji-on-blue-blocks.

## Changes

**Frontend**
- `index.css` ‚Äî full theme rewrite (tokens, typography, app shell, sidebar, transcript, tool cards,
  composer, approval card, token gate, pair modal, status). Subtle motion (message rise, status pulse).
- **Markdown rendering** (`Markdown.tsx`, `react-markdown` + `remark-gfm`) ‚Äî assistant replies render
  as headings/bold/lists/code/tables instead of raw `###` / `**`.
- **User vs assistant** ‚Äî user = right-aligned slate bubble; assistant = document with a copper rule
  + `claude` label.
- **Tool call summaries** (`tools.ts` `toolSummary`) ‚Äî the summary line shows the actual command/file
  (`Bash  pytest -x`, `Read  /repo/app.ts`) instead of just `üîß Bash`.
- **Composer** rebuilt ‚Äî a single bordered field with auto-grow textarea, ghost icon buttons
  (snippet/list/attach) and a circular accent send / stop button. (All `data-testid`s preserved.)
- Sidebar: brand, CSS status dots, repo-name titles, agent badge.

**Backend**
- `workspaces.ts` ‚Äî a non-repo root now **surfaces git repos one level down** (so `~/dreamLand/jodulabs`
  lists `trux`, `kalakar`, ‚Ä¶ instead of the bare dir). Fixes "shows /home/gp" *and* the git-noise root
  cause (you pick a real repo, so the agent's git probe doesn't `fatal`). git stderr also silenced.
- `pnpm pair` (`pair.ts` + `banner.ts`) ‚Äî prints the pairing QR + URLs and **exits 0**, so the QR isn't
  tied to a long-running foreground process. Env-loading + banner extracted from `index.ts`.
- `~/.trux/.env` `TRUX_WORKSPACES` repointed `$HOME` ‚Üí `/home/gp/dreamLand/jodulabs`.

**PWA**
- `sw.js` ‚Äî **network-first for navigations** (HTML always fresh ‚Üí new builds show up), cache-first
  only for hashed assets; cache bumped `v1`‚Üí`v2` to evict the stale shell. *This is why UI updates now
  appear after a refresh.*

## Verified
- Typecheck clean; **130 tests** (protocol 13 ¬∑ backend 84 ¬∑ frontend 33); build clean (fonts + markdown
  bundle); `pnpm pair` prints + exits 0.
- New tests: `toolSummary` (6), workspaces subdir-scanning (1).
